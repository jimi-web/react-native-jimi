# `Loading` 加载

## Static Methods
| Method | Params | Returns | Notes |
|---|---|---|---|
| show | 无| 无| 显示loading |
| hide | 无 |无| 隐藏loading |


## Demo
```

import {Circle} from  'react-native-jimi';

Circle.Loading.show();//显示
Circle.Loading.hide();//隐藏

```
