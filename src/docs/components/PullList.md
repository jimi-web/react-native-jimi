# `Loading` 加载

## Props
| Prop | Type | required | Default | Note |
|---|---|---|---|---|
| RefreshControl props | 无| 无 | 无| 继承RefreshControl的属性|
| hide | 无 | 无 | 无| 隐藏loading |


## Demo
```

import {Circle} from  'react-native-jimi';

Circle.Loading.show();//显示
Circle.Loading.hide();//隐藏

```
