# 几米圈标准小程序

## 描述
```
此项目为几米圈对几米小程序业务封装的标准化组件，其中包括定位，轨迹，追踪，录音，围栏等。
组件中包括业务功能和几米圈标准化接口，只适用于所有几米圈小程序开发者。
使用此组件能够实现快速开发，集成，迅速构建出一个具有功能性的小程序。
```

## 安装
```
1. npm install react-native-jimi --save

2.yarn add react-native-jimi
```
